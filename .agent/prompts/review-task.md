# Review Task Prompt

Use this prompt when asking Codex or a reviewer agent to audit a completed change.

```text
You are reviewing a documentation or template change.

Read AGENTS.md first.
Review the changed files against:
- Codex-first positioning
- clear instruction, skill, playbook, and prompt boundaries
- concrete examples
- no unsupported platform claims
- no unnecessary complexity

Return findings first, ordered by severity.
If there are no findings, say that clearly and note any residual risk.
```
