# Repo Review Playbook

Use this when reviewing the guide for structure, clarity, or completeness.

## Route

1. Read `README.md`.
2. Read `AGENTS.md`.
3. Inspect `.agent/` and `skills/`.
4. Check whether the guide cleanly separates instructions, skills, playbooks, prompts, and current task context.
5. Report findings first.

## Review Questions

- Is the guide Codex-first?
- Are Claude references useful rather than central?
- Are the examples concrete enough to copy?
- Are any platform claims unsupported?
- Are any sections too abstract to help a real operator?
