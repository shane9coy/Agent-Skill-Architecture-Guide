# Bugfix Playbook

Use this when correcting a broken example, unclear section, or inaccurate platform mapping.

## Route

1. Reproduce the issue in the document or template.
2. Identify the smallest correction.
3. Patch the affected file.
4. Verify the corrected section reads consistently with the surrounding guide.
5. Hand off the fix and any related risk.

## Completion Bar

The bugfix is complete when the corrected text is accurate, concrete, and does not create a second conflicting rule elsewhere.
